3d engine

3d graphics engine from scratch

The 3D engine was created in C++ from scratch in console. All of the math and calculations had to be created in order for the engine to work.

In this project we started with copying pixels, to drawing lines and triangles, converting that to 3d and then adding texturing and filtering.

This project was created with a hierarchical design. Important code was placed in a higher parent header file to allow access to other parts in the code.
